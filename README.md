# TikTok-live-Chat-reader-feedback

!NOTA: en el Directorio /python, se encuentran unos script base solo para ejecutarse en cosola

Esta aplicación es completamente gratuita y ha sido creada como un aporte personal para todos. La versión 2.0 trae mejoras en el voiceover para personas con discapacidad visual y actualiza el modelo de voz.

Ten en cuenta que es una versión abierta, y el plugin de TikTok que utilizamos podría ser bloqueado por la propia plataforma, así que te recomendamos usarlo con precaución.

Si tienes alguna sugerencia de mejora, no dudes en dejarnos tus comentarios desde la [pagina](https://momdontgo.dev/app_review). Y si te encanta, puedes apoyarnos con [donaciones](https://www.paypal.com/paypalme/dgbp00) o recomendándonos a otros.

Además, si estás interesado en un desarrollo personalizado,[ ¡no dudes en contactarme!](https://momdontgo.dev)

[Descarga](https://github.com/dimasb69/TikTok-live-Chat-reader-feedback/releases) 

Nota: Es posible que el instalador para Windows sea detectado como un virus debido a que el ejecutable de instalación no posee firmas registradas por Microsoft. Si lo prefieres, puedes optar por descargar el archivo .zip, que es una versión portátil. Ten en cuenta que todo el contenido del archivo .zip debe permanecer en la misma carpeta para que se reconozcan todas las librerías.
